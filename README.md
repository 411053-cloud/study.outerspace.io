# STUDY.SPACE

簡易專注計時器網站，包含：

- 高科技太空風 UI
- 專注計時任務
- 完成任務可獲得 CREDITS
- 用 CREDITS 解鎖飛船船體與尾焰外觀
- 使用 localStorage 保存進度與設定

開啟 `index.html` 即可體驗網站。

## GitHub Pages 部署

本專案可直接部署到 GitHub Pages。只要將此倉庫推送到 `main` 分支，並啟用 GitHub Pages，即可透過：

- `https://411053-cloud.github.io/study.outerspace.io/`

或查詢你的 GitHub Pages 網址。

部署方式：

1. 將變更 `git add .`、`git commit`、`git push` 到 `main`
2. 進入 GitHub 倉庫設定，確認 Pages 來源為 `main` 分支中的 `/ (root)`
3. 等待 Actions 自動部署完成。
